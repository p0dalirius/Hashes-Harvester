# Hashes-Harvester

Automatically extracts NTLM hashes from Windows memory dumps.

![](./.github/example.png)

## Usage

```
Hashes-Harvester: Automatically extracts NTLM hashes from Windows memory dumps.
Usage : winpass.sh MEMORYFILE
```
